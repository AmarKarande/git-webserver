🐳 Docker MySQL Setup (Step-by-Step from Pull)
Step 1: Pull MySQL Image
docker pull mysql:8

👉 Downloads MySQL image from Docker Hub

Step 2: Verify Image
docker images

✔ Check if mysql:8 is available

Step 3: Run MySQL Container
docker run -d --name my-mysql -e MYSQL_ROOT_PASSWORD=root123 -e MYSQL_DATABASE=cdac -v mysql-data:/var/lib/mysql -p 3306:3306 mysql:8

👉 OR multi-line:

docker run -d \
  --name my-mysql \
  -e MYSQL_ROOT_PASSWORD=root123 \
  -e MYSQL_DATABASE=cdac \
  -v mysql-data:/var/lib/mysql \
  -p 3306:3306 \
  mysql:8
Step 4: Check Running Container
docker ps

✔ Ensure container is running

Step 5: Check Logs (Important)
docker logs my-mysql

👉 Wait until:

ready for connections
Step 6: Connect to MySQL
docker exec -it my-mysql mysql -u root -p

Enter password:

root123
Step 7: Common Error Fix

❌ -e: command not found
✔ Reason: Multi-line without \
✔ Solution: Use \ or single-line command

Step 8: Remove Container (if needed)
docker rm my-mysql

If running:

docker rm -f my-mysql
🧠 Key Concept
Image = blueprint
Container = running instance
Always check logs before connecting
